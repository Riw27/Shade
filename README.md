# Shade
Запуск осуществляйте через [update.exe](https://github.com/Riw27/Shade/blob/main/update.exe?raw=true)
## Команды
1. ping
2. delme [кол-во сообщений] <ID | 0 - для удаления из всех активных ЛС>
3. editme [кол-во сообщений] [контент]
4. ban [ID] <причина>
5. slowmode <секунды>
6. user [ID]
7. copy
8. stop
9. restart
10. typing [секунды]
11. /cmd [команда]
12. /screenshot
